# Finance Dashboard UI (Zorvyn Assignment)

A clean, responsive finance dashboard built with React + JavaScript + Tailwind CSS.

## Features Implemented

- Dashboard overview with summary cards (balance, income, expenses)
- Time-based visualization (monthly trend chart)
- Categorical visualization (spending breakdown pie chart)
- Transactions list with date, amount, category, type, and description
- Search, filtering, and sorting in transactions
- Role-based UI (Viewer and Admin)
- Insights section (highest spending category and monthly comparison)
- State management with React hooks + reducer
- Dark/light mode toggle
- Local storage persistence
- Export to CSV and JSON
- Responsive layout for mobile and desktop

## Tech Stack

- React (JavaScript)
- Tailwind CSS
- Recharts
- Lucide React icons
- Vite

## Run Locally

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## Deploy to Vercel

1. Push this project to GitHub.
2. Open [Vercel](https://vercel.com/), import the GitHub repository.
3. Framework preset: `Vite`.
4. Build command: `npm run build`
5. Output directory: `dist`
6. Deploy.
